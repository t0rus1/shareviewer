Overview

The Overview presents a table with one entry per share. Each entry's contents is taken from the last time band of each share's All-Table, and therefore is also dependant on the values pertaining for the calculation parameters at the time they we generated.

Options available

  • Compile the Overview straight. Top right button
  • Recalculate the overview. Also top right button, but this will only happen if you've changed any parameter settings at the top.
  • You can at any stage do a Quick Save, which will save the state of the Overview table at that moment. The overview is saved in the All-Tables folder under the name overview.at and can at any stage be recalled using the Quick-reload
  • You can at any stage also save an Overview as a named Overview, with notes, and later reload to continue working with it
  • Since there are many columns in the table, you are allowed to select which columns you want to see. A particular set of columns is known as a View
    • .
  • You may name and save such view arrangements for later recall. It makes sense to name the views based on the calculation or topic you are focussing on.
  • You may open up any share's All-Table by double clicking in the left hand column
  • You may select one or individual rows (Ctrl-Click) for either deletion or saving. Saves may be named, and have notes.
  • There is a Save|Load drop-down button for this aforementioned purpose.
  • If you Right-Click in the left hand column, ShareViewer will show you the date of the last day of that All-Table. This is useful for confirming you are seeing what you expect to see.

How to Sort

The Overview allows for single column sorting. Simply click on a columns header to switch between ascending and descending order.

How to filter

Right Click on a column header if you wish to apply a filter criterion to a column.
A Filters box will pop up, with a filter condition row appended. Click in the Compare column to set a Greater Than or Less Than condition.
Note: in order to switch between these two conditions, just click again.
Click in the Value column to set a threshold value. Enable or disable the condition using the Apply check-box.
Add additional filter conditions for other columns by continuing to right click the relevant column headers, as described above...
Lastly, click the Apply button at the bottom. The Overview table should refresh, showing only those shares which meet your conditions.
